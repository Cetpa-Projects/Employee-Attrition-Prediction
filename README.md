# Employee Attrition Prediction

## Project Description

Employee Attrition Prediction is a Machine Learning-based project that predicts whether an employee is likely to leave the organization. The system analyzes factors such as salary, job satisfaction, performance, work environment, and experience to help companies improve employee retention and HR decision-making.
